# Guia de estudo Java

## Objetivo

Adquirir conhecimento sobre a linguagem Java, principais conceitos e stack usadas atualmente na empresa.

**Stack:**

- Java
- Spring Boot
- JPA + Hibernate
- Lombok
- JUnit

**Conceitos:**

- Principais:
    - POO (Programação Orientada a Objetos)
    - IoC (Inversion of Control)
    - DI (Dependency Injection)
    - JPA (Java Persistence API)
    - Spring (Framework de aplicações Java)
    - Hibernate (ORM Java)
    - JPA (Java Persistence API)

- Secundários:
    - Spring Data (Framework de persistência Java)
    - Spring Cloud (Framework de cloud Java)
    - Spring Batch (Framework de batch Java)
